Environmental Applications

How can computer science help our environment?

Technology can be used to create computer models using scientific data, these models can help environmentalists monitor changes to our planet.


Weather Forecasting

Using a combination of past and present data, climate models can be constructed to help us monitor current atmopsheric conditions and predict how weather conditions may change in the future. It is important for the weather to be monitored so that warnings can be issued if people and property are in danger. Satellite imagery allow us to view changes in weather patterns over large areas, using these satellites we can track tropical storms and predict where they will move to next.
